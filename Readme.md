## cordova-reactvr-testapp

React VR example app for cordova

### How to build apps

1. Install cordova

    ```
    $ npm install -g cordova
    ```
    
2. Add Platforms

    ```
    $ cordova platform add ios --save
    $ cordova platform add android --save
    ```
    
3. Build

    ```
    $ cordova build ios
    $ cordova build android
    ```
    
4. Run by emulator

    ```
    $ cordova emulate ios
    $ cordova emulate android
    ```
